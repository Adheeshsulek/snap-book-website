## Overview
This project is a web-based photo book creation platform that allows users to
upload images, select layouts, and customize photo arrangements dynamically.

The focus of the project is on frontend layout logic, image handling, and
user-friendly design for personalized photo books.

## Current Features
- Image upload functionality
- Multiple layout selection
- Basic zoom and positioning controls
- Frontend UI for photo placement

## Known Issues
- Zoom functionality currently applies to only the first uploaded image
- Layout option UI overlap needs refinement

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Canvas API
